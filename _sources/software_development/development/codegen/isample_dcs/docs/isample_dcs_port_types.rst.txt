*isample_dcs_port_types.h*
~~~~~~~~~~~~~~~~~~~~~~~~~~

Due to the nature of C++ the data types to be interchanged using msgpack need to be declared and defined at compile time.
This file includes the list of specific C++ data types required for this DCS data ports.
